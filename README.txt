En esta entrega se realizó un proyecto con componentes básicos de HTML
---------------------------------------------------------------------------
Forma de ejecución:
Debe tener instalada la extensión de Live Server en VSCode 

1) Abrir el repositorio de GitHbub
2) Copiar el código que aparece
3) Pegar el código en un archivo html de VSCode 
4) Hacer click derecho y apretar "Open with Live Server"
5) Y listo, debería aparecerle el proyecto de HTML
